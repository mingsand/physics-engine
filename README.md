# physics-engine
matter.js 라이브러리를 이용한  물리엔진
 wasd로 파란 사각형 조종

## 사용한 라이브러리

### Matter.js

- **이름**: [Matter.js](https://brm.io/matter-js/)
- **설명**: Matter.js는 2D 물리 엔진으로, JavaScript로 작성되었으며 HTML5 게임이나 애니메이션에 사용할 수 있습니다.
